# Proyecto Final – Simulador interactivo en JavaScript

Este proyecto simula un sistema de compra de productos, implementando un login de usuarios con roles.

## Funcionalidades
- Login de usuarios
- Roles: administrador y cliente
- Carrito de compras
- Simulación de compra
- Panel administrador para agregar productos
